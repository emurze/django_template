# Django Template

# How install project?

* ```git clone git@github.com:emurze/django_template.git```

  
* ```mv django_template/ <project_name>```


* ```cd <project_name>```


* ```bash setup.sh <project_name>```

# How run project?

Run dev server

```docker compose up --build```

Run prod server

```docker compose -f docker-compose.prod.yml up --build```
